## 📋 Что это?

**АвтоДКP** — система автоматизации оформления договоров купли-продажи недвижимости.

**Цель:** Олеся загружает фото документов → система распознаёт → проверяет → генерирует готовый ДКП → подаёт в Росреестр.

## 🚀 Деплой

### Render.com (рекомендуется)
1. Пуш на GitHub: `./push_to_github.sh`
2. Создать сервис на Render.com
3. Указать `BOT_TOKEN` в Environment Variables
4. Готово!

### Docker локально
```bash
docker-compose up -d
# Веб: http://localhost:8501
# Бот: работает в фоне
```
