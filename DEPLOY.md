# 📱 АвтоДКP — Запуск ТОЛЬКО с телефона (без компьютера)

## 🎯 Цель
Запустить бота на бесплатном хостинге, чтобы он работал 24/7. Всё делается с телефона через браузер.

---

## 🚀 Способ 1: Replit (ПРОСТОЙ — рекомендуется)

### Шаг 1: Регистрация (2 минуты)
1. Открой браузер телефона → зайди на **replit.com**
2. Нажми **Sign Up** (зарегистрироваться)
3. Выбери **Continue with Google** → выбери свой Google-аккаунт
4. Подтверди email (зайди в Gmail → найди письмо от Replit → нажми подтвердить)

### Шаг 2: Создание проекта (2 минуты)
1. Нажми **Create** (зелёная кнопка +)
2. Выбери **Python** (иконка с змеёй)
3. В поле **Title** напиши: `autodkp-bot`
4. Нажми **Create Repl**

### Шаг 3: Загрузка файлов (3 минуты)
1. Слева нажми на **⋮** (три точки) рядом с Files
2. Нажми **Upload file**
3. Загрузи файл `telegram_bot_unified.py` из архива (из папки `bot/`)
4. Загрузи файл `backend.py` из архива (из папки `core/`)

### Шаг 4: Создание .env (1 минута)
1. Слева нажми **New file** (иконка +)
2. Назови файл: `.env` (точка в начале!)
3. Напиши:
   ```
   BOT_TOKEN=здесь_твой_токен
   ```

### Шаг 5: Получение токена (3 минуты)
1. Открой Telegram на телефоне
2. Найди контакт **@BotFather**
3. Отправь: `/newbot`
4. Ответь на вопросы:
   - Имя бота: `AutoDKP`
   - Username: `autodkp_твоеимя_bot` (обязательно `_bot` в конце!)
5. BotFather пришлёт токен — длинная строка цифр и букв
6. **Скопируй токен** → вставь в файл `.env` вместо `здесь_твой_токен`

### Шаг 6: Установка библиотек (1 минута)
1. Снизу найди вкладку **Shell** (иконка >_)
2. Вставь команду:
   ```bash
   pip install aiogram==2.25.1 python-dotenv
   ```
3. Нажми Enter → жди пока установится

### Шаг 7: Запуск (1 минута)
1. Нажми зелёную кнопку **Run** сверху
2. Бот запущен! ✅

### ⚠️ Важно: Replit "засыпает"
- Бот остановится через 15 минут если никто не заходит
- **Решение:** Открой сайт **uptimerobot.com**
  - Зарегистрируйся
  - Нажми **Add New Monitor**
  - Тип: HTTP(s)
  - URL: скопируй ссылку из Replit (нажми **Open in a new tab** → скопируй адрес)
  - Интервал: 5 минут
  - Нажми **Create Monitor**
- Теперь бот будет просыпаться каждые 5 минут!

## 🚀 Способ 2: Render.com (РАБОТАЕТ 24/7)

### Шаг 1: Подготовка
1. Зайди на **github.com** → зарегистрируйся
2. Создай новый репозиторий: нажми **+** → **New repository**
3. Название: `autodkp-bot`
4. Нажми **Create repository**

### Шаг 2: Загрузка файлов
1. Нажми **uploading an existing file**
2. Перетащи файлы:
   - `bot/telegram_bot_unified.py`
   - `core/backend.py`
   - `requirements.txt`
   - `render.yaml`
3. Нажми **Commit changes**

### Шаг 3: Подключение к Render
1. Зайди на **render.com** → зарегистрируйся через GitHub
2. Нажми **New** → **Web Service**
3. Выбери свой репозиторий `autodkp-bot`
4. Нажми **Connect**

### Шаг 4: Настройка
1. Название: `autodkp-bot`
2. Runtime: **Python 3**
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `python bot/telegram_bot_unified.py`
5. Нажми **Advanced** → добавь переменную:
   - Key: `BOT_TOKEN`
   - Value: твой токен от BotFather
6. Нажми **Create Web Service**

### Шаг 5: Готово!
- Бот работает 24/7!
- Можно закрыть сайт — бот не остановится

## 📱 Как пользоваться ботом

После запуска:
1. Открой Telegram
2. Найди своего бота (по username из шага 5)
3. Отправь `/start`
4. Создавай сделки!

**Пример использования:**
```
📱 Телефон (в дороге):
   → Зашёл в бота → создал сделку
   → Ввёл ФИО продавца
   → Сфоткал паспорт (пришлёт фото боту)
   → Сохранил

💻 Компьютер (дома):
   → Открыл того же бота в Telegram Web
   → Продолжил ту же сделку!
   → Дописал адрес и цену
   → Сгенерировал договор

📱 Телефон (снова):
   → Проверил — всё на месте
   → Подписал договор
   → Готово!
```

## 🆘 Если что-то не работает

| Проблема | Решение |
|----------|---------|
| Бот не отвечает | Проверь токен в `.env` |
| Replit остановился | Настрой UptimeRobot |
| Ошибка в консоли | Перезапусти (нажми Stop → Run) |
| Не могу найти бота | Проверь username (должен оканчиваться на `_bot`) |

## 💡 Советы

- **Сохрани ссылку на Replit** в закладки — чтобы быстро перезапускать
- **Сохрани токен** в заметках телефона — он не меняется
- Для постоянной работы используй **Render.com** (не засыпает)
