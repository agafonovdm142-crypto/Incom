## 📱 ЧАСТЬ 1: Telegram-бот

### 1.1 Первый запуск

**Найдите бота:** @AvtoDKP1_bot в Telegram или перейдите по ссылке.

**Отправьте:** `/start`

```
┌─────────────────────────────┐
│  🤖 АвтоДКP                 │
│                             │
│  Привет, Олеся!            │
│                             │
│  🏠 Автоматизация договоров│
│     купли-продажи          │
│                             │
│  [➕ Новый ДКП]            │
│  [📋 Мои сделки]           │
│  [⚙️ Настройки]            │
│  [❓ Помощь]               │
└─────────────────────────────┘
```

### 1.2 Создание сделки (6 шагов)

**Нажмите:** ➕ **Новый ДКП**

| Шаг | Действие | Ввод |
|-----|----------|------|
| 1 | Фото паспорта продавца | 📷 или ✏️ ручной ввод |
| 2 | Фото паспорта покупателя | 📷 или ✏️ ручной ввод |
| 3 | Адрес квартиры | Текст или 📷 выписка ЕГРН |
| 4 | Цена | Число (аванс 4% авто) |
| 5 | Банк эскроу | Кнопка: Сбер/ВТБ/Альфа |
| 6 | Проверка и генерация | ✅ Подтвердить |

**Формат ручного ввода:**
```
ФИО;Дата рождения;Серия;Номер;Кем выдан;Дата выдачи;Адрес

Пример:
Иванов Иван Иванович;01.01.1990;45 06;123456;ОВД ЦАО;10.02.2010;г. Москва, ул. Ленина 1
```

### 1.3 Готовый договор

После шага 6 бот отправляет PDF:
```
🎉 Договор готов!

📄 DKP-2026-0043.pdf (245 KB)

Содержит:
• Договор купли-продажи
• Передаточный акт
```

## 💻 ЧАСТЬ 2: Веб-приложение

### 2.1 Запуск

**Локально:** http://localhost:8501  
**Облако:** https://autodkp.onrender.com

### 2.2 Главная страница

- 📋 **Мои сделки** — список с фильтрами и поиском
- ➕ **Новая сделка** — 6-шаговый мастер
- ⚙️ **Настройки** — профиль, банки, уведомления
- ❓ **Помощь** — инструкции и FAQ

### 2.3 Детали сделки (4 вкладки)

| Вкладка | Содержимое |
|---------|-----------|
| 📄 Договор | Текст договора, проверка, скачивание PDF |
| 👤 Стороны | Редактирование данных продавца/покупателя |
| 🏠 Объект | Адрес, кадастр, площадь, загрузка ЕГРН |
| 💰 Расчёты | Цена, аванс, банк эскроу, проверка сумм |

### 2.4 Проверка расчётов (авто)

```
Цена:      12 500 000 ₽
Аванс:        500 000 ₽
Основной:  12 000 000 ₽
─────────────────────────
✅ Сходится: 500 000 + 12 000 000 = 12 500 000
```

Если суммы не сходятся — красное предупреждение.

## 🔄 Синхронизация 📱↔💻

```
1. 📱 Создаёте сделку в боте
2. 💻 Открываете веб — сделка уже там!
3. 💻 Редактируете детали на компьютере
4. 📱 Проверяете в боте — всё обновилось!
```

**Как это работает:** данные хранятся в `deals.json` (локально) или PostgreSQL (в облаке).

## 🆘 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Бот не отвечает | Отправьте `/reset`, затем `/start` |
| OCR не распознал | Сделайте фото при лучшем освещении |
| Расчёты не сходятся | Проверьте, что Аванс + Основной = Цена |
| Веб не загружается | Проверьте, что порт 8501 свободен |
| Ошибка импорта | Установите зависимости: `pip install -r requirements.txt` |
